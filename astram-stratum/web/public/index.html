Quick Start

Start mining in 3 steps

Get your GPU contributing in under 5 minutes.

Download the miner

Download the Astram node for your OS. It includes the GPU miner and wallet tools. An NVIDIA GPU with 4 GB+ VRAM is required for pool mining.

FAQ

Frequently Asked Questions

What GPU do I need?

Any NVIDIA GPU with at least 4 GB VRAM is supported. The algorithm uses a 4 GB memory-hard DAG, so AMD GPUs and CPUs are not currently supported. GTX 1650 Super, RTX 2060, and newer cards work well.

How is the block reward distributed?

The pool uses PPLNS (Pay Per Last N Shares) with a window of 10,000 shares. When a block is found, the 8 ASRM block reward (minus 1% pool fee) is distributed proportionally to everyone who contributed shares in the window. Miners with more shares in the window earn more.

What is the minimum payout?

Balances accumulate in your account until you request a withdrawal. There is no automatic minimum threshold — you decide when to withdraw via the wallet-cli tool.

How do I check my balance?

Visit the Pool Dashboard and search for your address in the Miners table. You can also check via the API: GET http://pool.Astramchin.com:8081/payments.

What is the block time?

Astram targets a 2-minute (120-second) block time. Difficulty adjusts every block using the DWG3 algorithm, so the block time stays stable regardless of network hashrate changes.

Is there a stratum port for high-difficulty miners?

Currently all miners connect to port 3333. The pool uses VarDiff, automatically adjusting your work difficulty based on your GPU's hash rate. No manual configuration needed.

Where is the source code?

The full node, miner, pool server, and wallet are all open source at github.com/YSBlueA/AstramChain.
